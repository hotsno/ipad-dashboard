# iPad Dashboard
Lazy vibecoded dashboard thingy.

I run this on [Asobi](https://github.com/kingbri1/Asobi) on my iPad under my monitor (see pics below!)

> [!TIP]
> You can use iPad/iOS shortcuts app to open `asobi://ipad-dashboard-gamma.vercel.app`, then place this shortcut on your home screen
